Select a driver and enter the start date and end of the report time range.
You can customize the report to get a weekly report, monthly and / or details and you can select the activity types which
will be shown in the report. After entering data click the printer icon   and the report generation will start.
The different meaning of the times shown in the report is the following:
Daily Driving: Driving time
Other Works: Working time
Availability: Availability time
Indeterminate: Indeterminate time
Rest: Break / Rest time
