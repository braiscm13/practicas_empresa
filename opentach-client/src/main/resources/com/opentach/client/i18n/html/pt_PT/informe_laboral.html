Seleccionar um condutor e introduzir a data de início e de finalização do período para o qual se pretende obter o relatório. Pode-se
personalizar o relatório para obter um relatório semanal, mensal e/ou com detalhes e pode-se seleccionar os tipos de actividades
das quais se pretenda obter o relatório. Uma vez introduzidos os dados clique no ícone da impressora   e será iniciada a preparação do relatório.
O significado dos tempos mostrados no relatório é o seguinte:

Condução Diaria: Tiempo de condução
Otros Trabalhos: Tiempo de trabalho
Disponibilidade: Tiempo de disponibilidad
Indefinido: Tiempo indefinido
Descanso: Tiempo de pausa / descanso
