Cet écran vous permet d'obtenir les infractions des 28 derniers jours d’une entreprise. Cliquez sur
le bouton  ou directement sur le bouton pour obtenir le rapport. Cette opération peut prendre quelques minutes.
Une fois l'analyse réalisée, vous pouvez personnaliser le rapport. Après avoir entré les données,
cliquez sur l'icône de l'imprimante   et l’élaboration du rapport commencera.
