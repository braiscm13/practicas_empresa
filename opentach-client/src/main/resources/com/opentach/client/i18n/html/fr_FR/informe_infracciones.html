Cet écran vous permet d'obtenir les infractions d’un conducteur donné dans une période de temps stipulé en cliquant sur
le bouton  ou directement sur le bouton pour obtenir le rapport. Cette opération peut prendre quelques minutes.
Une fois l'analyse réalisée, vous pouvez personnaliser le rapport en montrant les détails et en sélectionnant les types d’infractions à montrer dans le rapport.
Après avoir entré les données, cliquez sur l'icône de l'imprimante   et l’élaboration du rapport commencera.
