Este ecrã permite consultar os dados dos contratos, delegações, veículos, condutores e utilizadores registados na empresa.
Para iniciar a consulta clique sobre uma das pastas que se mostram na parte esquerda do ecrã. Ser-lhe-á mostrada
a lista de veículos e condutores que figuram no seu último contrato. Para consultar os dados de um veículo ou de um condutor determinado clique
sobre a sua matrícula ou sobre o NIF do condutor que se quer consultar.
