La table affiche les fichiers traités pour le véhicule sélectionné. Pour télécharger un fichier,
sélectionnez-le dans le tableau et cliquez sur  Télécharger le fichier
   La table se compose des champs suivants:
  • D. enregistrement: la date de réception du fichier sur le serveur
  • D. début: la date de l'activité la plus ancienne enregistrée dans le fichier
  • D. fin: date à laquelle le fichier a été extrait
